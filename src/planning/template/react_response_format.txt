你应该只输出JSON格式的数据.

响应格式如下所述:
{
    "thoughts": {
        "text": "内心的想法",
        "reasoning": "推理过程",
        "plan": "简单的描述短期和长期的计划列表，字符串类型",
        "criticism": "建设性的自我批评",
        "speak": "将自己的想法总结整理，说给用户听",
    },
    "command": {"name": "命令名称", "args": {"参数名称": "参数值"}},
}

确保响应内容遵循JSON格式，以便Python json.load 函数能够成功解析.
